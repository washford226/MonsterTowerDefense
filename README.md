# Monster Tower Defense

A 2D tower defense game where the player controls monsters defending against invading knights.

## Features
- Archer towers with projectile system
- Melee units that block and fight enemies
- Wave-based enemy system
- Fully testable combat and targeting systems

## Tech
- Unity (C#)
- NUnit (Unity Test Framework)
- GitHub

## Goal
Demonstrate QA-focused development with unit testing, system design, and debugging tools.
